# 第 0 章: 从零开始 —— 完全没写过 Python 也能看懂

这一章是给**完全没接触过编程 / Python** 的同学准备的热身。看完你就知道:
一行 Python 是怎么"跑起来"的、怎么让它把结果显示出来、怎么调用一个功能。

> 💡 **边看边敲**: 左下角有一个「🐍 试一试」交互框,敲一行代码按回车就能立刻看到结果,
> 就像一个计算器。下面每个例子都建议你亲手在那里敲一遍——**看十遍不如敲一遍**。

## 1. 最简单的一行: 让 Python 说话

Python 里最常用的一个功能叫 `print`,作用是把括号里的东西"打印"出来(显示在屏幕上):

```python
print("你好, Python!")
```

在交互框里敲上面这行、按回车,你会看到下面显示:

```
你好, Python!
```

- `print` 是功能的名字(叫**函数**)。
- 后面的一对**圆括号** `()` 表示"调用"这个功能。
- 括号里面 `"你好, Python!"` 是**给它的东西**(叫**参数**)。

一句话总结调用方法的套路: **名字( 要处理的东西 )**。这个套路你后面会见到无数次,比如
`pd.read_csv("data/orders.csv")`,其实就是"调用 read_csv 这个功能,把文件名交给它"。

## 2. 引号: 文字要用引号包起来

上面 `"你好, Python!"` 两边有引号。**一段文字(叫字符串)必须用引号包起来**,
单引号 `'...'` 和双引号 `"..."` 都行:

```python
print('也可以用单引号')
```

如果不加引号,Python 会以为你在说一个"名字",找不到就会报错。数字则**不用**加引号:

```python
print(2024)
```

## 3. 数字可以直接算

Python 本身就是个强大的计算器,常见运算:

```python
print(3 + 5)      # 加,得到 8
print(10 - 4)     # 减,得到 6
print(6 * 7)      # 乘(用星号 *),得到 42
print(20 / 3)     # 除,得到 6.666...
```

`#` 后面的内容是**注释**,是写给人看的说明,Python 会自动忽略,不影响运行。

## 4. 变量: 给数据起个名字

你可以把一个值"存"起来,给它起个名字,之后重复使用。这就是**变量**,用 `=` 赋值:

```python
name = "勇士"
age = 18
print(name)          # 勇士
print("年龄是", age)  # 年龄是 18
```

- `=` 在编程里是"把右边的值存进左边这个名字",不是数学里的"相等"。
- `print` 括号里可以放多个东西,用逗号 `,` 隔开,会挨个打印出来(中间自动加空格)。

## 5. 一切皆可"点方法"

除了 `print` 这种独立的函数,很多东西自带一些"技能",用 **点 `.`** 来调用,
这叫**方法**。比如字符串自带一个把字母变大写的技能 `upper`:

```python
word = "hello"
print(word.upper())   # HELLO
```

读法:`word` 这个字符串,点一下、调用它的 `upper()` 方法。后面 pandas 里
`customers.head()`、`orders.groupby(...)` 全都是这个"**东西.方法()**"的写法。

## 6. 看看一个东西是什么类型

`type()` 能告诉你某个值是什么类型,初学时很有用:

```python
print(type(123))       # <class 'int'>   整数
print(type(3.14))      # <class 'float'> 小数
print(type("hi"))      # <class 'str'>   字符串
print(type(True))      # <class 'bool'>  布尔(真/假)
```

## 小结

- 让 Python 显示结果用 `print(...)`。
- 调用功能的套路是 **名字(参数)**;对象自带的技能用 **对象.方法()**。
- 文字要加引号,数字不用;`#` 是注释;`=` 是赋值(起名字)。
