# 使用方法

这个项目有两种学习方式,你可以任选其一,或者混着用。

## 方式一: 命令行(适合已经装好 Python 的人)

1. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```
2. 生成练习用的数据(会在 `data/` 目录下生成 3 张"表"的 CSV):
   ```bash
   python data/generate_data.py
   ```
3. 按顺序学习每一章:
   - 先读该章目录下的 `notes.md`(概念 + SQL 对照 + 代码示例)
   - 再打开 `exercises.py`,把标了 `# TODO` 的地方补全
   - 补完后直接运行这个文件,脚本会自动校验你的答案对不对:
     ```bash
     python 02_pandas_basics/exercises.py
     ```
   - 卡住了就看同目录下的 `solutions.py` 参考答案(建议先自己写,写不出来再看)

校验输出格式(每道题不管对错都会打印预期和实际,方便对照):

```
[ex3_filter_over_100]
预期: [120.5, 199.0, 599.0]
实际: [120.5, 199.0, 599.0]
结果: Pass!
```

答对时命令行里也会掉金币、赛利亚也会给你鼓励:

```
通过 10 / 10 题
✨ 💰💰💰💰💰💰💰💰💰💰  金币 x10 掉落!✨
[赛利亚] 全部通过!勇士,你离传说中的数据分析师又近了一步!
```

## 方式二: 网页学习模式(推荐,尤其是 Python 小白)

不想开编辑器、甚至还没装 Python 的话,可以直接在浏览器里边看笔记边写代码运行
(基于 Pyodide,Python 真的跑在你的浏览器里,首次加载需要联网下载运行环境,约 10~30 秒)。

### Windows 用户: 双击 bat 一键启动

如果你是 Windows 10,直接双击项目根目录的 **`启动网页学习.bat`** 就行。
它会自动:

1. 检查有没有装 Python,没装的话自动帮你装(优先用系统自带的 winget,
   装不了会引导你到官网下载)
2. 在项目根目录启动本地服务器(`python -m http.server 8000`)
3. 自动打开浏览器访问 <http://localhost:8000/web/>

> 第一次运行如果需要安装 Python,可能会弹出用户账户控制(UAC)让你确认,点"是"即可。
> 装完 Python 后,如果 bat 提示找不到 python,关掉窗口重新双击一次 bat 就好
> (因为新装的 Python 要重开命令行窗口才会生效)。

### 其他系统 / 手动启动

在项目根目录启动一个静态服务器:

```bash
python -m http.server 8000
```

然后浏览器打开 <http://localhost:8000/web/> 。

### 网页里都有什么

- **完全没写过 Python?** 先看第 0 章「Python 入门」,教你 `print()`、怎么调用方法、变量,
  零基础也能跟上。
- **中间**是每章的笔记(SQL 对照),点左侧栏切换章节;不想看笔记专心做题时,点工具栏的
  **「📖 收起笔记」** 可以把中间这栏整个收起来,再点一次展开。
- **中间栏底部**有一个 **🐍 试一试** 交互框:敲一行 Python 回车就立刻出结果
  (像 `>>>` 命令行那样),边看笔记边把例子敲一遍,不用先写整个文件。点它的标题栏可收起。
- **右边**是代码编辑器和输出窗口:
  - 编辑器带 **Python 语法高亮**(关键字、字符串、注释、数字都会上色)
  - 右上角 **🌙 / ☀️ 按钮**可以切换白天 / 暗黑模式(会记住你的选择)
  - `data/` 下的三张表已经自动加载进去,`pd.read_csv("data/orders.csv")` 直接可用
  - 快捷键 **Ctrl+Enter** 运行

### 网页里的正反馈系统(星星进度制)

- 点 **「📘 载入本章练习」** 会把该章的 `exercises.py` 直接载入编辑器,补全 TODO 后点运行。
- **每道题第一次做对会点亮一颗星**(有星星动画和音效);重复通过不再重复加,
  所以工具栏和左侧栏会显示你每章的进度,比如 `⭐ 3/6`,**全部点亮(🏆)就算这一章过关**。
- 右下角的 **赛利亚** 会根据你的运行结果说话:点亮新星夸你、满星恭喜过关、
  部分对鼓励你、报错了教你看报错信息(台词会随机变化)。
